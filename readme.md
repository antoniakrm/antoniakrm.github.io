## Personal Website

Static personal website for Antonia Karamolegkou.

### Main Pages

- `index.html` - homepage with profile and selected publications
- `publications.html` - full publications list
- `activities.html` - talks, workshops, and outreach
- `mycat.html` - cat gallery page

### CV / Resume

- Current resume file: `AntoniaK_Resume.pdf`
- The top navigation on all main pages links to this file.

If you update the resume again, replace `AntoniaK_Resume.pdf` in the repo root (or update the nav links accordingly).

### Local Preview

Open `index.html` directly in a browser, or run a local static server from the project root:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
